qbm.utils.load_raw_data(data_dir, data_source, start_date=datetime.datetime(1999, 1, 1, 0, 0), end_date=datetime.datetime(2019, 12, 31, 0, 0), min_volume=1, common_dates=True, drop_holidays=True)

Loads the raw csv files from the data directory. The files must corresponds to the strings in the currency_pairs list, i.e., if currency_pairs = [“abc”, “xyz”], then the files in the data directory must contain “abc” and “xyz” in their names.

Parameters

  • data_dir – Directory containing the raw csv files.

  • currency_pairs – List of strings corresponding to the file names (w/o extensions).

  • start_date – Date at which to drop all data before (inclusive).

  • end_date – Date at which to drop all data after (inclusive).

  • min_volume – Filter out dates with volume less than the provided min_volume.

  • common_dates – Ensure all dataframes have the same dates.

  • drop_holidays – Drop dates that are NYSE or LSE holidays.

Returns

Dictionary with the strings in currency_pairs as keys and dataframes as values.
